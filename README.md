# LexiTide Legal Pages

## About LexiTide

LexiTide is a mobile word puzzle game where players connect letters, discover words, collect coins, use hints, find bonus words, build their Word Journal, and complete relaxing word levels.

## Publisher

**Naresh Game Studio**
